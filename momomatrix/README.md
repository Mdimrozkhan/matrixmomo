# MomoMatrix - Future of Momos

A modern Next.js website for MomoMatrix momo business with a striking black and red theme. Built with TypeScript, Tailwind CSS, and featuring cutting-edge animations and user interactions.

## 🚀 Features

- **Modern Design**: Black and red color scheme with circuit board patterns
- **Responsive Layout**: Mobile-first design that works on all devices
- **Interactive Pages**: Home, Menu, Offers, and Reel & Win sections
- **Live Features**: Real-time order counter and customer statistics
- **Animations**: Smooth transitions, glowing effects, and hover states
- **Social Integration**: Instagram Reel & Win campaign
- **SEO Optimized**: Meta tags and semantic HTML

## 📋 Pages

- **Home**: Hero section with brand messaging and features
- **Menu**: Interactive menu with pricing and add-to-cart functionality
- **Offers**: Special combo deals and discount information
- **Reel & Win**: Social media campaign with reward tiers

## 🛠 Tech Stack

- **Framework**: Next.js 16 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Fonts**: Geist Sans & Geist Mono
- **Icons**: SVG icons and emojis

## 🎨 Theme

- **Primary Colors**: Black (#000000) and Red (#dc2626)
- **Accent Colors**: Gray tones for depth
- **Animations**: Glow effects, pulse animations, and smooth transitions
- **Typography**: Modern, clean fonts with good readability

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

```
src/
├── app/
│   ├── menu/          # Menu page
│   ├── offer/         # Offers page
│   ├── reel-win/      # Reel & Win page
│   ├── globals.css    # Global styles and theme
│   ├── layout.tsx     # Root layout
│   └── page.tsx       # Home page
├── components/
│   ├── Navigation.tsx # Site navigation
│   ├── Footer.tsx     # Site footer

│   └── LiveCounter.tsx # Live statistics
```
